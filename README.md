# FantaRunning


Author: MCED(mced.it)

Based on: 
[Assemble.io](http://assemble.io)
Grunt, Bower, Libsass

## For working: how to setup / launch server

- clone & cd /TOT
- $ bower install && npm install
- $ grunt server 
- go to localhost:9000


## How to deploy

- take all /dist content and put it on your webserver :)
